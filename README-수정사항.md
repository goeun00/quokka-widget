# Quokka Work Dock 수정사항

## 🎯 주요 수정 내용

### 1. 개발자 도구 유지
- `main.js`: 개발자 도구 주석 해제 (항상 활성화)

### 2. 캐릭터 선택 기능 추가
- **설정에 캐릭터 선택 추가**: 쿼카 🐨 / 토끼 🐰 중 하나만 Dock에 표시
- 선택한 캐릭터만 하단 Dock에 표시됨
- **둘 다 같은 패널 열기**: 어떤 캐릭터를 선택하든 동일한 패널이 열림 (이미지만 다름)
- 설정은 localStorage에 저장되어 유지됨

### 3. 사용자 이름 설정
- **설정에서 사용자 이름 입력 가능**: @goeun 대신 원하는 이름 표시
- 헤더 타이틀에 `@사용자이름 · Jira` 형식으로 표시
- localStorage에 저장되어 유지됨

### 4. 브랜치 관리 개선
- ❌ **제거**: "저장된 브랜치가 없어요" 안내 문구
- ❌ **제거**: 취소 버튼 (저장 버튼만 유지)
- ✅ **개선**: 빈 값으로 등록 시도 시 경고 메시지 표시
- 🎨 **UI 개선**: 
  - "추가" 버튼 → Edit 아이콘(✏️)으로 변경
  - 클릭 시 Edit 아이콘 → Check 아이콘(✓)으로 변경
  - 입력창에 저장(✓) 버튼만 표시
  - ESC 키로 편집 모드 종료 가능
  - Enter 키로 저장

### 5. 링크 관리 개선
- ✅ **링크 이름 수정 가능**: URL뿐만 아니라 링크 표시 이름도 수정 가능
- 두 개의 입력 필드:
  - 링크 이름 (표시명)
  - URL (주소)

### 6. 메모 관리 개선
- ❌ **제거**: "메모를 추가해보세요" 안내 문구
- 🎨 **UI 개선**:
  - **우측 상단 버튼 방식**: Edit 아이콘(✏️) → 편집 모드에서 체크(✓) / 취소(×)로 변경
  - 메모 인풋창 아래 버튼 제거
  - 체크 버튼: 메모 저장 및 편집 모드 종료
  - 취소 버튼: 변경사항 버리고 편집 모드 종료

### 7. 레포지토리 관리 개선
- ✅ **레포지토리 수정 가능**: 수정 버튼(✏️) 추가
- 수정 버튼 클릭 시:
  - 해당 레포 정보가 입력 필드에 자동 입력됨
  - 목록에서는 제거됨
  - 수정 후 다시 추가하면 저장됨

## 📝 파일별 변경사항

### main.js
- 개발자 도구 활성화

### index.html
- 설정 섹션에 "사용자 이름" 입력 필드 추가
- 설정 섹션에 "Dock Character" 선택 추가

### renderer.js
- `state.dockCharacter` 추가
- `state.userName` 추가
- `updateDockVisibility()` 함수 추가
- `bindMemoToggle()` 함수 추가
- 토끼/곰 버튼 모두 같은 패널 열기로 변경
- 사용자 이름 반영 (헤더 타이틀)
- 브랜치 추가 UI/이벤트 개선 (취소 버튼 제거)
- 메모 우측 상단 버튼 방식으로 변경
- 링크 이름 수정 기능 추가
- 레포지토리 수정 기능 추가
- 설정 로드/저장 시 캐릭터 선택 및 사용자 이름 처리

### style.css
- 메모 편집 액션을 우측 상단에 배치
- 메모 인풋 아래 액션 버튼 제거
- 링크 입력 필드 레이아웃 수정 (이름 + URL)
- 브랜치 추가 행 그리드 조정 (2칸)
- 레포 액션 스타일 추가

## 🚀 적용 방법

1. 기존 파일 백업
2. 수정된 파일들로 교체:
   - `main.js`
   - `index.html`
   - `renderer.js`
   - `style.css`
3. 나머지 파일들은 기존 것 유지 가능:
   - `github.js`, `jira.js`, `preload.js`, `store.js`

## ✨ 사용자 경험 개선사항

- 불필요한 안내 문구 제거로 깔끔한 UI
- 빈 값 입력 방지로 데이터 품질 향상
- 명확한 저장/취소 버튼으로 사용자 의도 확실하게 반영
- 캐릭터 선택으로 개인화된 경험 제공 (이미지만 변경)
- 사용자 이름 설정으로 더욱 개인화된 인터페이스
- 링크 이름 수정으로 더 직관적인 링크 관리
- 레포지토리 수정 기능으로 편리한 관리
- 메모 편집 UI를 우측 상단 버튼으로 통일하여 일관성 향상
